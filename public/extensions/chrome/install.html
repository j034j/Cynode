Install the Cynode Chrome Bridge

This extension lets the Cynode web app import open tabs, recent bookmarks, and browser history from Chrome.

  1. Open chrome://extensions in Chrome.
  2. Turn on Developer mode.
  3. Click Load unpacked.
  4. Select the folder that contains this extension's manifest.json.
  5. Return to the Cynode web app, open Import from Browser, and click Detect Extension.

If you are working from this repo locally, the extension files live beside this page:
