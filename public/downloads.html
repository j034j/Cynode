Install Cynode Desktop on Windows, macOS, or Linux

Cynode Desktop gives you the same account-backed workspace as the web app, but with a dedicated installed application, persistent sign-in, a reusable in-app viewer, and desktop shortcuts that launch straight into Cynode.

Windows

Recommended for most users. Install once, sign in once, and then launch Cynode Desktop from the Start Menu or desktop shortcut.

  • Primary target: Windows installer built with NSIS.
  • Also available: portable Windows executable for no-install environments.
  • After first launch, Windows registers the cynode:// handler for deep links from the web app.

macOS

Configured for DMG and ZIP release packaging so Mac users can install Cynode Desktop as a native app.

  • Targets configured: dmg and zip.
  • Architectures configured: Intel and Apple Silicon.
  • For signed public distribution, the final release build should be produced on macOS.

Linux

Configured for AppImage, Debian packages, and tarballs so Linux users have a straightforward install path too.

  • Targets configured: AppImage, deb, and tar.gz.
  • Architectures configured: x64 and arm64 where supported.
  • Best for users who want a local desktop launcher without keeping a browser tab open.
