Desktop App

Launch Cynode as a full desktop workspace

Cynode Desktop mirrors the same account-backed experience as the web app, and now includes a persistent in-app viewer that can load node URLs inside a reusable desktop browsing window instead of spawning endless browser tabs.

Browser Bridge
Install is required before launch. The browser can only open cynode:// after Cynode Desktop has been installed or at least started once on this device.
  1. Download and install Cynode Desktop.
  2. Open Cynode Desktop once so Windows registers the cynode:// handler.
  3. Come back here and use Launch Installed Desktop App.

What syncs

  • Your Cynode account, saved graphs, shares, media, and analytics sync through the same backend.
  • Desktop and web stay aligned when you sign into the same Cynode account.
  • Browser cookies are not imported automatically from Chrome, Edge, or Firefox.

Deep Link Launcher

The installed Electron app registers cynode://, so the web app can ask the OS to launch the desktop app with user consent. If Windows says there is no registered handler, the app is not installed yet.
